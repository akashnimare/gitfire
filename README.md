## Installation
npm install --global gitfire

## Usage

> gitfire commitmessage
## What It Does

- creates new branch `fire`
- adds all files
- commits with `"firefire"` or custom message
- pushes to remote

## Usage

`gitfire <commitmessage>`

`<commitmessage>` is optional. If not specified, `"firefire"` will be used.
